I)Introduction

Overview

Hewo is excellent for a newspaper, magazine or publishing site. This template built with HTML5, CSS3 and Bootstrap system. Hewo is fully responsive, it looks stunning on all types of screens and devices.

We have a dedicated support center for all of your support needs. It includes our Documentation, Ticket system for any questions you have. We usually get back to you within 12-24 hours.

Features

  • HTML5 / CSS 3
  • Fully Responsive (Tested on Multiple Devices)
  • Smart responsive navigation
  • Offcanvas sidebar
  • Full width or Boxed Layout
  • 05 Home page layouts, 04 Archive page layouts, 05 post format layout (video, gallery, audio, image, text)
  • Carousel, Slideshow news
  • Free Google Web Fonts
  • Free Icon Fonts (FontAwesome)
  • Color & Fonts is easily changed
  • W3C Validated Code
  • 21 HTML files
  • HTML5 and CSS3, Bootstrap 4.x system
  • Compatible Browsers: IE8+, Firefox, Safari, Opera, Chrome
  • Well documentation included
  • Ticket System Support
